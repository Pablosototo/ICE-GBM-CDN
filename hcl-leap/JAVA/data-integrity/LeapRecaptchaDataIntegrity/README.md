# Lean Recaptcha Data Integrity (HCL Leap)

Proyecto Java OSGi (Data Integrity) para validación de Google reCAPTCHA en HCL Leap.

## Tecnologías
- Java
- OSGi
- Eclipse PDE
- HCL Leap

## Dependencias
Este proyecto requiere librerías propietarias de HCL Leap.
Por razones de licencia, dichas librerías no se incluyen en este repositorio.

Para compilar:
1. Obtener las librerías desde el servidor HCL Leap
2. Colocarlas en una carpeta `lib/`
3. Agregarlas al Build Path en Eclipse

## Tipo de proyecto
OSGi Bundle (Eclipse Plug-in Development Environment)
